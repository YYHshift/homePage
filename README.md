# Personal Homepage Repository

The project source files now live inside `./site`. For setup, tech stack, and deployment notes, please read [`site/README.md`](./site/README.md).

If you only need the ready-to-run Next.js project, clone this repo and jump directly into the `site/` directory:

```bash
cd site
npm install
npm run dev
```

All documentation, including SQLite data maintenance and deployment steps, is maintained in that inner README.
